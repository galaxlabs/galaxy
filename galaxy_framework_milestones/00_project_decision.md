# Galaxy Framework — Project Decision

## Current Direction

We are building the framework logic in Python first.

Go support is not cancelled. Go will come later after the Python core proves the architecture.

## Current Temporary Name

Product / Framework: Galaxy Framework  
CLI: galaxy  
Desk/UI: Galaxy Desk  
Builder: Galaxy Studio  
Cloud/Bench Manager: Galaxy Cloud / Bench Manager  

Do not use the old Gogal name in this Python project. Gogal was for the Go/Golang direction.

## Main Idea

Galaxy Framework is a metadata-driven business application framework.

It is inspired by Frappe/Odoo/Tryton/Django Admin ideas, but it must become its own framework.

## Core Flow

Bench CLI
→ Site setup
→ Core metadata
→ DocType Builder
→ Migration Preview
→ Apply Migration
→ Dynamic CRUD API
→ Dynamic Desk List/Form
→ Permissions
→ Security hardening
→ Bench Manager / Cloud Lite
→ SaaS support
→ Go runtime support later
