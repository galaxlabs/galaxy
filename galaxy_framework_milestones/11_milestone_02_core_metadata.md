# Milestone 2 — Core Metadata

## Goal

Make the framework understand its own metadata.

## Build

- DocType metadata table
- DocField metadata table
- DocPerm metadata table
- Error Log metadata/table
- core metadata seed

## Core DocTypes

- Installed App
- Installed Module
- Module Def
- DocType
- DocField
- DocPerm
- User
- Role
- Has Role
- Error Log

## Important Rule

Business DocTypes like Customer/Supplier/Invoice are not part of this milestone.

## Acceptance Criteria

- core metadata tables exist
- core DocType records exist
- core DocField records exist
- core DocPerm records exist
- doctor validates metadata
- tests pass
